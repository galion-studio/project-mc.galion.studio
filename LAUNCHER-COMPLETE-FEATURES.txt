HOW TO USE:
-----------

BASIC PLAY:
1. Open launcher
2. Enter username
3. Click "INSTALL & PLAY" (first time)
4. Wait for download
5. Minecraft opens automatically!

WITH FORGE (MODS):
1. Click Forge checkbox
2. Follow popup instructions
3. Download from files.minecraftforge.net
4. Install Forge
5. Add mods to 'mods' folder
6. Launch via official Minecraft launcher's Forge profile

WITH OPTIFINE (PERFORMANCE):
1. Click OptiFine checkbox
2. Follow popup instructions
3. Download from optifine.net
4. Run OptiFine installer
5. Launch via official Minecraft launcher's OptiFine profile

TROUBLESHOOTING:
----------------

"97% stuck forever!"
  → It's NOT stuck! Asset download is slow.
  → Wait 5-10 minutes total
  → Check network activity in Task Manager
  → See: WHY-97-PERCENT-HANGS.txt

"Launch Failed"
  → Install Java 17+: java.com
  → Update graphics drivers
  → Restart computer
  → Try again

"Can't connect to server"
  → Make sure server is running: docker-compose ps
  → Check address: localhost:25565
  → Use Java Edition (not Bedrock!)

Forge not working?
  → Use official Minecraft launcher's Forge profile
  → Don't use Galion launcher for modded

OptiFine not working?
  → Use official Minecraft launcher's OptiFine profile
  → Don't use Galion launcher for OptiFine
