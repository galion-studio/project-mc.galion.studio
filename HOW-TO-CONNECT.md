## â“ Frequently Asked Questions

### Is GALION compatible with Minecraft?
No, GALION is a completely separate platform. It's built from scratch with its own engine, though it shares some similar gameplay concepts.

### Do I need to buy Minecraft?
No! GALION is 100% free and doesn't require Minecraft or any other purchase.

### Can I use Minecraft mods?
No, but GALION has its own plugin/mod system. We're building a community marketplace for GALION-specific mods.

### Is the AI always listening?
Voice features are push-to-talk by default. You control when the AI listens. Privacy settings are fully configurable.

### Can I play offline?
The AI features require an internet connection, but we're working on offline modes with local AI models.

### How is my data used?
GALION is open source and privacy-focused. Voice data is processed for AI responses but not stored long-term. Read our privacy policy for details.

### Can I contribute to development?
Absolutely! Check out our GitHub repository and join our Discord to get involved.

## ğŸ› ï¸ Troubleshooting

### Client won't launch
1. Check system requirements
2. Update graphics drivers
3. Run as administrator (Windows)
4. Check antivirus isn't blocking it

### Can't connect to server
1. Check internet connection
2. Verify server address: `mc.galion.studio`
3. Check firewall settings
4. Try restarting the client

### Voice features not working
1. Check microphone permissions
2. Test microphone in system settings
3. Verify mic is selected in GALION settings
4. Check internet connection (AI needs connection)

### Low performance/FPS
1. Lower graphics settings
2. Close other applications
3. Update graphics drivers
4. Check system meets minimum requirements
