==========================================
GALION STUDIO MINECRAFT LAUNCHER
==========================================

Welcome to the official Galion Studio custom Minecraft launcher!

REQUIREMENTS:
-------------
- Minecraft Java Edition (installed)
- Internet connection

WINDOWS USERS:
--------------
1. Double-click "GalionLauncher.exe"
2. Enter your Minecraft username
3. Click the "PLAY" button
4. Wait for Minecraft to start
5. Go to Multiplayer in Minecraft
6. Connect to: mc.galion.studio
7. Enjoy!

LINUX USERS:
------------
1. Open terminal in launcher folder
2. Make executable: chmod +x GalionLauncher
3. Run: ./GalionLauncher
4. Enter your Minecraft username
5. Click the "PLAY" button
6. Wait for Minecraft to start
7. Go to Multiplayer in Minecraft
8. Connect to: mc.galion.studio
9. Enjoy!

FEATURES:
---------
✓ Simple and clean interface
✓ Remembers your username
✓ Quick launch to our server
✓ Lightweight and fast

TROUBLESHOOTING:
----------------
Problem: "Minecraft not found"
Solution: Make sure Minecraft is installed
          and you can launch it normally first.

Problem: Launcher won't start
Solution: Make sure you extracted all files.
          On Windows, try right-click > "Run as Administrator"

Problem: Can't connect to server
Solution: Make sure you're using Minecraft Java Edition
          (not Bedrock). The server address is:
          mc.galion.studio
